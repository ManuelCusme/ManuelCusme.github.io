Sobre mí

Soy Manuel Cusme, desarrollador Full Stack con base en Ecuador 🇪🇨. Me apasiona construir software que resuelve problemas reales, con especial interés en la arquitectura de sistemas y el desarrollo backend robusto.

Mi stack principal gira alrededor de Java para el backend, JavaScript/TypeScript para el frontend, y también trabajo con Python y C# dependiendo del proyecto. Creo en el poder del código limpio y las metodologías ágiles.

Actualmente estudio y construyo proyectos que van desde juegos en Java con arquitecturas por capas hasta aplicaciones web full stack, siempre buscando aprender y mejorar cada iteración.
